# Dashboard Template - Elzero Template Four

A modern, responsive dashboard template built with HTML and Vanilla CSS. This project is part of the Elzero Web School web development path, focusing on complex layouts, responsive design, and organized CSS architecture.

## ✨ Features
- **Responsive Design:** Optimized for all screen sizes (Desktop, Tablet, and Mobile).
- **Custom CSS Framework:** Built using a modular utility-first CSS approach.
- **8+ Unique Pages:**
    - **Dashboard:** Overview widgets with statistics and progress tracking.
    - **Settings:** Extensive user configuration options and toggles.
    - **Profile:** User info, skills, and activities.
    - **Projects:** Interactive project list with status indicators.
    - **Courses:** Grid layout for available educational content.
    - **Friends:** Social cards with contact and stats information.
    - **Files:** File manager layout with different file type previews.
    - **Plans:** Subscription and pricing tier comparisons.
- **Rich UI Components:** Progress bars, custom toggle switches, responsive tables, and interactive widgets.
- **Aesthetic UI:** Uses a clean, modern color palette and smooth transitions.

## 🛠️ Built With
- **HTML5:** Semantic structure.
- **CSS3:** Custom properties and Flexbox/Grid layouts.
- **Font Awesome 6:** For high-quality iconography.
- **Google Fonts:** "Open Sans" for clean typography.
- **Normalize.css:** For consistent cross-browser rendering.

## 📁 Project Structure
```text
├── css/
│   ├── all.min.css       # Font Awesome
│   ├── framework.css    # Custom utility classes
│   ├── normalize.css    # CSS Reset
│   └── style.css        # Main project styles
├── imgs/                # UI assets and avatars
├── index.html           # Dashboard Page
├── settings.html        # Settings Page
├── profile.html         # Profile Page
├── projects.html        # Projects Page
├── courses.html         # Courses Page
├── friends.html         # Friends Page
├── files.html           # Files Page
└── plans.html           # Plans Page
```
